# 布置运行环境：

好的让我们来布置环境运行这段代码吧！

首先下载一个JDK，点击此处跳转到下载界面 (opens new window)

点击下载最下面那个Windows x64 Installer 159.71 MB后面的下载得到一个安装文件(.exe)

然后什么都不用改，直接无脑下一步即可

提示下载完后找到安装位置(一般为C盘的Program File\Java\jdk-xx.x.x)

复制这个目录，鼠标右键此电脑，选择属性→高级系统设置→环境变量，找到系统变量那一栏(用户变量下面那一栏)新建变量，第一行命名为JAVA_HOME,第二行粘贴刚才复制的目录，点击确定

按win键+R键(win键一般在Alt左边那个windos图标的按键)输入cmd

写入命令java、javac、java -version都有反应而不是出现'javac' 不是内部或外部命令，也不是可运行的程序或批处理文件这种提示即可

# 软件安装

接下来是软件

我这里是用的Vscode这款软件，因为写C用，懒得换大众化的Eclipse IDE,

所以此教程为用Vscode编写Java，如果您想用非Vscode编写，请移步百度搜索您软件的使用教程

首先找到Vscode的官网：点击这里 (opens new window)

点击中间那个蓝色按钮：Download for Windows

下载完毕后安装

出现一个选择在进行软件安装的时候是否同时进行其他任务，这里全部选择，然后继续下一步直至安装完成

# 设置中文版VSCode

可能你们下载的是英文版，以下步骤可设置为中文

  • 启动VSCode，然后按Ctrl + Shift + P打开命令调试板，然后输入“Configure Display Language”，选择并按下Enter键，会打开一个locale.json文件。将文件中的配置项修改为"locale":"zh-cn"，并且保存修改。【新版不需要此步骤，直接进行下一步即可】
  • 接着在VSCode拓展商店安装所需要的语言包，在搜索框中输入“Language Packs”，选择中文（简体）安装。
  • 最后重启VSCode，就可以开始使用中文版啦。

  1. Code Runner

此插件是运行代码的关键，必装

  1. Java Extension Pack

Java的一个必装插件，以下从3到6这4个插件都是第二个自带的，所以不用下载，缺哪一个下哪一个就成

  1. Debugger for Java
  2. Java Test Runner
  3. Maven for Java
  4. Project Manager for Java

配置完成

此时再翻上去复制一下上面那个HelloWorld，在文件夹里新建一个文本文档，改后缀名为.java，全名为HelloWorld.java，在文件夹下点击左上方文件按钮，出现Windows powershell，鼠标移上出现以管理员身份打开Windows powershell，在命令行输入

javac HelloWorld.java
1

无错误的话它会跳到下一行并在你HelloWorld的文件夹内出现一个后缀名为.class的文件

此时用Vscode打开HelloWorld.java的话它会在文件开头出现一个红色圆形“J”标志

点击右上角一个三角型的“Run Java”

点击运行即可出现“HelloWorld”

好的，教程结束！
